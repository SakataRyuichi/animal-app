### WEB-007: プライバシーポリシー表示
**As a** アプリのプライバシー保護について知りたいユーザー  
**I want to** プライバシーポリシーを確認できる  
**So that** 個人情報の取り扱いについて理解できる

**受け入れ基準**:
- プライバシーポリシーの全文を表示
- 改定日時を表示
- バージョン管理：改定履歴を表示
- SEO最適化：適切なメタタグを設定
- LLMフレンドリー：構造化された形式で記述

**体験価値**:
- **透明性**: 個人情報の取り扱いが明確で、信頼性が向上
- **法的対応**: 法的要件を満たし、リスクを低減

**使用シーン**:
- アプリをダウンロードする前にプライバシーポリシーを確認したい時
- 個人情報の取り扱いについて疑問を持った時

**優先度**: 最高（法的要件）

### WEB-010: 法務ドキュメントの管理（管理者）
**As a** 管理者  
**I want to** プライバシーポリシー、利用規約、特定商取引法表記などの法務ドキュメントを管理できる  
**So that** 法務要件を満たし、必要に応じて更新できる

**受け入れ基準**:
- Convexの`legal_documents`テーブルに法務ドキュメントを追加・編集できる
- ドキュメントタイプ（プライバシーポリシー、利用規約、特定商取引法表記など）を選択できる
- Markdown形式での記述に対応
- バージョン管理：改定履歴を保持
- 改定日時を自動記録
- ユーザーへの通知：改定時にアプリ内で通知を表示

**体験価値**:
- **法的対応**: 法務要件を満たし、リスクを低減
- **効率性**: 簡単に法務ドキュメントを更新できる
- **透明性**: 改定履歴を保持し、透明性が向上

**使用シーン**:
- 法務ドキュメントを初回作成する時
- 法務ドキュメントを改定する時
- 新しい法務要件に対応する時

**優先度**: 最高（法的要件）
