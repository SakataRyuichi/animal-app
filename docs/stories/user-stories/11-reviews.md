## レビュー機能

#### US-048: レビュー投稿 ✅ **2026年更新 - 餌のレビューに特化した詳細情報**
**画面パス**: `app/products/reviews/create.tsx`

**As a** 商品を使用した飼い主  
**I want to** 商品にレビューを投稿し、特に餌の場合は成分表や栄養成分に関する情報も共有できる  
**So that** 他のユーザーに商品の評価と詳細な使用感を共有できる

**受け入れ基準**:
- **基本情報**:
  - 評価（1-5）を選択できる
  - コメントを入力できる
  - 使用したペットを選択できる
  - 投稿後、商品のレビュー一覧に表示される
- **餌のレビュー専用フィールド**（オプション）:
  - **成分表の確認**: 成分表を確認したかどうかを選択できる（「確認済み」「未確認」）
  - **栄養成分の評価**: 栄養成分についての評価を選択できる（「適切」「やや不足」「不足」など）
  - **使用期間**: 使用期間を入力できる（例: 「1ヶ月」「3ヶ月」「1年以上」）
  - **使用量**: 1日の使用量を入力できる（例: 「100g」「200g」）
  - **ペットの反応**: ペットの反応を選択できる（例: 「喜んで食べる」「普通」「あまり食べない」）
  - **健康への影響**: 健康への影響を選択できる（例: 「体調が良くなった」「変化なし」「体調が悪くなった」）
- **レビューの公開設定**:
  - レビューを公開するかどうかを選択できる（デフォルト: 公開）
  - 非公開の場合は、自分のみが閲覧可能

**体験価値**:
- **詳細性**: 特に餌のレビューでは、成分表や栄養成分に関する情報も共有でき、より詳細な評価ができる
- **信頼性**: 使用期間や使用量などの情報により、レビューの信頼性が向上する
- **参考性**: ペットの反応や健康への影響などの情報により、他のユーザーが商品を選ぶ際の参考になる
- **包括性**: 基本的な評価から詳細な使用感まで、幅広い情報を共有できる

**使用シーン**:
- 新しいフードを試した後、成分表を確認してレビューを投稿
- 長期間使用したフードについて、使用期間や健康への影響を含めてレビューを投稿
- ペットの反応が良かったフードについて、詳細な情報を含めてレビューを投稿

**優先度**: 高

#### US-050: 種別ごとの人気商品ランキング
**画面パス**: `app/products/ranking.tsx`
**As a** 自分のペットに最適な商品を探している飼い主  
**I want to** 特定の種別に人気の商品ランキングを見る  
**So that** 同じ種別のペットを飼っている多くの飼い主が選んでいる商品を知り、信頼性の高い商品を選べる。特に珍しい種別（爬虫類、小動物など）でも、同じ種別の飼い主の評価を参考にできる

**受け入れ基準**:
- 種別を簡単に選択でき、自分のペットに合わせたランキングをすぐに見られる
- レビュー数や評価順でランキングが表示され、信頼性の高い商品が上位に来る
- 「フトアゴヒゲトカゲに人気のフード」「シベリアンハスキーにおすすめのフード」など、種別・品種で絞り込める
- 各商品にレビュー数、平均評価、価格帯が表示され、比較しやすい
- 「この商品を使っている飼い主」が表示され、同じ種別の飼い主の使用実績が分かる
- 商品をタップすると詳細ページに遷移し、レビューや詳細情報を確認できる

**体験価値**:
- **信頼性**: 同じ種別の多くの飼い主が選んでいる商品を知り、安心して選べる
- **効率性**: 自分で商品を探す手間が省け、時間を節約できる
- **包括性**: 珍しい種別でも、同じ種別の飼い主の評価を参考にできる

**使用シーン**:
- 新しいフードを探している時、同じ種別のペットに人気の商品を確認する
- 初めてペットを飼う時、先輩飼い主が選んでいる商品を参考にする
- 珍しい種別のペットを飼っている時、同じ種別の飼い主の評価を参考にする

**優先度**: 最高（Phase 3の核心機能）
