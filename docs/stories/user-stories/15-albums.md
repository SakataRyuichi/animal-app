## アルバム管理機能

#### US-060: アルバム作成
**画面パス**: `app/albums/create.tsx`
**As a** ペット飼い主  
**I want to** 日記や写真をテーマ別にアルバムにまとめる  
**So that** 思い出を整理し、後から振り返りやすくなる

**受け入れ基準**:
- アルバム作成画面で、タイトル（例：「初めてのドッグラン」「5歳の誕生日」）と説明を入力できる
- 表紙画像を選択できる（オプション）
- アルバムを作成できる
- **無料ユーザー**: 最大2つまで作成可能
- **プレミアムユーザー**: 無制限に作成可能
- 制限数に達している場合、「プレミアムならアルバムを無制限に作れます」という温かいアップグレード案内が表示される

**体験価値**:
- **思い出の整理**: テーマ別に思い出を整理できる
- **愛着の深まり**: アルバムを作ることで、ペットとの時間を大切に感じられる

**使用シーン**:
- 特定のイベント（誕生日、初めてのお散歩など）の思い出をまとめたい時
- 通院記録をまとめて管理したい時

**優先度**: 高

#### US-061: アルバムに日記・写真を追加
**画面パス**: `app/albums/[id].tsx`
**As a** ペット飼い主  
**I want to** 日記や写真をアルバムに追加する  
**So that** 思い出をテーマ別に整理できる

**受け入れ基準**:
- 日記詳細画面の隅に「＋」アイコンが表示される
- 「＋」アイコンをタップすると、アルバム選択シートが下から表示される
- 複数のアルバムにチェックを入れるだけで追加完了
- 「新しいアルバムを作る」ボタンも表示される
- 日記一覧から「複数選択」して一括でアルバムに追加できる「整理モード」が利用できる
- **無料ユーザー**: 1アルバム20枚まで追加可能
- **プレミアムユーザー**: 1アルバム無制限に追加可能
- 制限数に達している場合、「プレミアムならアルバムを無制限に追加できます」という案内が表示される

**体験価値**:
- **直感的な操作**: 日記から簡単にアルバムに追加できる
- **効率的な整理**: 複数選択で一括追加できる

**使用シーン**:
- 過去の日記を振り返っている時、アルバムに追加したい時
- 複数の日記を一度にアルバムに追加したい時

**優先度**: 高

#### US-062: アルバム一覧・詳細表示
**画面パス**: `app/albums/index.tsx`, `app/albums/[id].tsx`
**As a** ペット飼い主  
**I want to** 作成したアルバムの一覧を見る  
**So that** テーマ別に整理した思い出を振り返れる

**受け入れ基準**:
- アルバム一覧画面で、作成したアルバムが表示される
- アルバムの表紙画像、タイトル、説明が表示される
- アルバム内のアイテム数が表示される
- アルバムをタップすると、アルバム詳細画面に遷移する
- アルバム詳細画面で、追加した日記や写真が時系列で表示される
- アルバム内のアイテムの順序を変更できる（ドラッグ&ドロップ）

**体験価値**:
- **思い出の振り返り**: テーマ別に整理された思い出を振り返れる
- **整理の実感**: アルバムを作ることで、思い出が整理されていることを実感できる

**使用シーン**:
- 特定のテーマの思い出を振り返りたい時
- アルバムを整理したい時

**優先度**: 高

#### US-063: アルバムの共有・送信（プレミアム限定）
**画面パス**: `app/albums/[id].tsx`
**As a** プレミアム会員  
**I want to** アルバムを共有・送信する  
**So that** 親戚や友人と思い出を共有できる

**受け入れ基準**:
- **Webアルバム生成**: 特定のアルバムを「期間限定のWebサイト（閲覧専用URL）」として発行できる
- 閲覧専用URLをSNSで共有できる
- URLは一定期間（例：30日）で無効になる
- **PDF書き出し**: アルバムにまとめた日記と写真を、そのまま一冊のフォトブックのようなレイアウトでPDF化できる
- PDFをメールで自分宛に送信できる
- PDFをダウンロードできる
- アルバム作成前にプレビュー画面が表示され、「どの写真を入れるか」「背景の色はどうするか」を選べる
- 「〇〇ちゃんの1才のあゆみ」というタイトルが自動で入る

**体験価値**:
- **思い出の共有**: 親戚や友人と思い出を共有できる
- **思い出の保護**: PDFとして手元に残せる安心感

**使用シーン**:
- ペットの誕生日パーティーのアルバムを親戚に共有したい時
- ペットとの思い出をPDFとして手元に残したい時

**優先度**: 高

#### US-064: アルバムの共同編集（プレミアム限定）
**画面パス**: `app/albums/[id].tsx`
**As a** プレミアム会員  
**I want to** 家族でアルバムを共同編集する  
**So that** 家族みんなで写真を出し合ってアルバムを作れる

**受け入れ基準**:
- アルバムに共同編集者を追加できる
- 共同編集者は、アルバムに日記や写真を追加できる
- 共同編集者は、アルバムのタイトルや説明を編集できる
- 共同編集者の一覧が表示される
- 共同編集者を削除できる（アルバム作成者のみ）

**体験価値**:
- **家族の協力**: 家族みんなで思い出を共有できる
- **思い出の充実**: 一人では撮れなかった写真も、家族で協力して集められる

**使用シーン**:
- 家族でペットの誕生日パーティーのアルバムを作りたい時
- 家族でペットとの思い出を共有したい時

**優先度**: 中
