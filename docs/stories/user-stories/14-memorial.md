#### US-057: 虹の橋を渡る（メモリアルモードへの移行）
**画面パス**: `app/settings/memorial.tsx`, `app/pets/[id]/index.tsx`
**As a** ペット飼い主  
**I want to** ペットが虹の橋を渡ったことを記録する  
**So that** ペットとの思い出を大切に保護できる

**受け入れ基準**:
- ペット詳細画面から「虹の橋を渡る」という特別なボタンにアクセスできる
- 命日を入力できる（オプションで、飼い主からの最後の一言も入力可能）
- メモリアルモードに移行すると、記録を「入力」するボタンが消え、代わりにこれまでの思い出を「振り返る」ボタンに変わる
- ペットのアイコンに、優しく光る輪や淡い背景色が添えられる
- 年齢表示は命日で固定される（例：「14歳5ヶ月でお空へ」）
- ペットは削除されず、いつでも振り返ることができる

**体験価値**:
- **思い出の保護**: ペットとの思い出が消えることなく、いつでも振り返ることができる
- **温かみ**: 「削除」ではなく「虹の橋を渡る」という優しい表現で、ペットとの時間を大切にできる
- **安心感**: 記録が消えることなく、いつでも会いに来られる場所として機能する

**使用シーン**:
- ペットが虹の橋を渡った時
- ペットとの思い出を大切に保護したい時

#### US-058: 思い出のアルバム作成・エクスポート
**画面パス**: `app/albums/create.tsx`
**As a** プレミアム会員  
**I want to** ペットの思い出をアルバムとして書き出す  
**So that** アプリが消えても、思い出が手元に残る安心感を得られる

**受け入れ基準**:
- プレミアム限定機能として提供される
- 指定した期間（1ヶ月、1年、全期間など）の記録と写真を選択できる
- 以下の形式でエクスポート可能:
  1. **シンプルな画像共有**: 1枚の画像をSNS/メッセージで共有（無料ユーザーも利用可能）
  2. **デジタル・フォトアルバム（PDF）**: 指定期間の記録と写真をレイアウトしたPDFファイル（プレミアム限定）
  3. **一括ダウンロード（ZIP）**: 最高画質WebPをまとめてZIP圧縮してダウンロード（プレミアム限定）
- アルバム作成前にプレビュー画面が表示され、「どの写真を入れるか」「背景の色はどうするか」を選べる
- 「〇〇ちゃんの1才のあゆみ」というタイトルが自動で入る
- メールで自分宛に送信できる
- 処理が終わったらプッシュ通知やメールで「ダウンロードの準備ができました」と知らせる
- メモリアルモードに移行した際や、退会を検討した際に、「これまでの思い出を、一生消えない宝物として書き出しましょう」と優しく提案される

**体験価値**:
- **安心感**: アプリが消えても、思い出が手元に残る安心感
- **思い出の保護**: ペットとの思い出を「一冊の本」として残せる
- **プレミアムの価値**: サブスクリプションの「誠実な価値」として機能する

**使用シーン**:
- ペットが虹の橋を渡った後、思い出をアルバムとして残したい時
- 退会を検討している時、これまでの思い出を手元に残したい時
- ペットの成長記録を振り返りたい時

**優先度**: 高

#### US-059: メモリアルモードでの振り返り
**画面パス**: `app/pets/[id]/index.tsx`（メモリアルモード時）
**As a** ペット飼い主  
**I want to** メモリアルモードに移行したペットの思い出を振り返る  
**So that** ペットとの時間を大切に感じられる

**受け入れ基準**:
- メモリアルモードに移行したペットの詳細画面では、記録を「入力」するボタンが消え、代わりに「振り返る」ボタンが表示される
- これまでの記録（食事、トイレ、散歩、日記など）を時系列で振り返ることができる
- 写真をアルバム形式で閲覧できる
- 年齢表示は命日で固定される（例：「14歳5ヶ月でお空へ」）
- ペットのアイコンに、優しく光る輪や淡い背景色が添えられる

**体験価値**:
- **思い出の保護**: ペットとの思い出が消えることなく、いつでも振り返ることができる
- **温かみ**: 「いつでも会いに来られる場所」として機能する

**使用シーン**:
- ペットが虹の橋を渡った後、思い出を振り返りたい時
- ペットとの時間を大切に感じたい時

**優先度**: 高

#### US-067: 追悼（メモリアル）プランの導入 ✅ **2026年最終設計検証で追加**
**画面パス**: `app/settings/memorial.tsx`, `app/settings/premium.tsx`
**As a** ペットが虹の橋を渡った飼い主  
**I want to** 新しい記録がなくなった後も、思い出（高画質画像）を見るために課金を続けなくても良い  
**So that** 経済的な負担を感じることなく、ペットとの思い出を大切にできる

**受け入れ基準**:
- ペットが虹の橋を渡り、メモリアルモードに移行した際、「追悼（メモリアル）プラン」への移行を提案される
- メモリアルプランは月額無料または極安価で、データの保持と閲覧だけを許可する「読み取り専用」の状態
- プレミアム会員だった期間のデータは、退会後も一定期間（例：1年間）「最高画質」でエクスポート可能
- メモリアルプランでは、新しい記録の追加はできないが、これまでの記録と写真は最高画質で閲覧可能
- メモリアルプランから通常のプレミアムプランに戻ることも可能（新しいペットを追加する場合など）

**体験価値**:
- **経済的配慮**: 新しい記録がなくなった後も、経済的な負担を感じることなく思い出を大切にできる
- **思い出の保護**: プレミアム会員だった期間のデータは、退会後も一定期間最高画質でエクスポート可能

**使用シーン**:
- ペットが虹の橋を渡り、メモリアルモードに移行した時
- プレミアム会員だったが、新しい記録がなくなったため退会を検討している時

**優先度**: 中
