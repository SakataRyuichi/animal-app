#### US-008: 食事記録
**画面パス**: `app/pets/[id]/activities/feeding.tsx`  
**As a** 毎日ペットに食事を与えている飼い主  
**I want to** 食事内容を簡単に記録する  
**So that** 食欲の変化や食事量の推移を把握し、健康状態の早期発見につなげられる。また、かかりつけの獣医師に相談する際に正確な情報を提供できる

**受け入れ基準**:
- 3タップ以内で記録完了できる（種別・量・商品名の入力が簡単）
- **前回の入力内容を保持し、デフォルトで入力できる** ✅ **2026年追加**
  - 前回記録した餌の商品名、量（g）、商品IDが自動的に入力欄に表示される
  - 前回の記録がない場合のみ、手動で入力する
  - 前回の内容をそのまま使用する場合は、商品名と量を確認して記録ボタンをタップするだけ
  - 前回の内容を変更する場合は、商品名や量を編集してから記録できる
  - 前回の記録から一定期間（例: 7日）経過した場合は、デフォルト値として使用しない（古い情報のため）
- よく使う商品は履歴から選択でき、入力の手間が最小限
- 写真を添付して視覚的な記録も残せる（食欲不振時の比較に有効）
- メモで「いつもより食べが悪い」「完食した」などの観察を記録できる
- 記録後、タイムラインに表示され、過去の記録と比較しやすい
- 食事時間のリマインダーを設定でき、記録忘れを防げる

**体験価値**:
- **効率性**: 前回の入力内容が自動的に表示されるため、毎回同じ餌を与えている場合は記録がさらに簡単になる ✅ **2026年追加**
- **健康管理**: 食欲の変化を数値で把握でき、病気の早期発見につながる
- **安心感**: 獣医師に相談する際、正確な情報を提供でき、適切な診断を受けられる
- **継続性**: 簡単な操作で毎日の記録が習慣化できる

**使用シーン**:
- 朝・夕の食事タイムにスマホでサッと記録（前回の内容が自動入力されているため、確認して記録するだけ）
- 食欲が落ちた時に過去の記録と比較
- 新しいフードに切り替えた時の記録（前回の内容を編集して記録）
- 毎日同じ餌を与えている場合、前回の内容をそのまま使用して素早く記録 ✅ **2026年追加**

**優先度**: 最高
