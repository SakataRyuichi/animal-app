# 公式サイトユーザーストーリー

**📚 ドキュメントインデックス**: [DOCUMENTATION_INDEX.md](../../DOCUMENTATION_INDEX.md)

## 概要
このドキュメントは、公式サイト（Next.js + Vercel）の機能をユーザー視点で整理したユーザーストーリーです。

**関連ドキュメント**:
- [USER_STORIES.md](./USER_STORIES.md): モバイルアプリ側のストーリー
- [ADMIN_USER_STORIES.md](./ADMIN_USER_STORIES.md): 管理画面側のストーリー
- [CONVEX_SCHEMA.md](../schema/CONVEX_SCHEMA.md): スキーマ定義（news, legal_documentsテーブルなど）
- [DESIGN_DOCUMENT.md](../design/DESIGN_DOCUMENT.md): アプリ設計の詳細

各ストーリーは「As a / I want to / So that」の形式で記述され、SEOとLLMフレンドリーな構成、アプリのダウンロード導線、ニュース更新、法務ドキュメント管理を目的としています。

**このドキュメントは「開発の憲法」として機能します。**
CursorなどのAI開発ツールに機能の実装を依頼する際、このドキュメントを参照することで、SEOとLLMフレンドリーな公式サイトを作り出すことができます。

## トップページ・機能紹介

### WEB-001: トップページ表示
**As a** アプリを探している潜在ユーザー  
**I want to** アプリの概要と主要機能を一目で理解できる  
**So that** アプリの価値を素早く把握し、ダウンロードに進める

**受け入れ基準**:
- **メインビジュアル（ヒーローセクション）** ✅ **2026年追加 - ブランド戦略**
  - 構図：スマートフォンの中でアプリを操作する手元と、その奥でリラックスして眠るペットの「ボケ感」を活かした写真
  - メッセージ：「記録は、愛した証。」のような、情緒的で短いコピーを配置
  - 「このアプリがある生活がいかに安心で、温かいか」をブランドとして伝える
- **ストーリーテリング型のコンテンツ構成** ✅ **2026年追加 - ブランド戦略**
  - **Empathy（共感）**: 「今日、何回トイレに行ったっけ？」「この子の小さな変化、見逃したくない。」という飼い主の不安に寄り添う
  - **Solution（解決）**: アプリの機能（種類別記録、リマインダー）を、実際の操作画面アニメーションで提示
  - **Future（未来）**: 「虹の橋」や「アルバム」の機能を見せ、このアプリが一生（あるいは一生を超えて）寄り添う存在であることを提示
  - **Social Proof（信頼）**: バッジを付けた誇らしいペットたちのギャラリーや、最新のアップデートニュース
- ダウンロードボタン（App Store / Google Play）をヘッダーと各セクションに配置
- 主要機能の概要を視覚的に分かりやすく表示
- SEO最適化：適切なメタタグ、OGPタグ、構造化データ（JSON-LD）を設定
- LLMフレンドリー：Semantic HTML（article, section, navなど）を正しく使用

**体験価値**:
- **検索性**: SEO最適化により、検索エンジンから自然流入が期待できる
- **理解しやすさ**: 一目でアプリの価値が分かる
- **アクション性**: ダウンロードボタンが目立つ位置に配置され、コンバージョンが向上

**使用シーン**:
- 検索エンジンから公式サイトにアクセスした時
- SNSでアプリの紹介を見て公式サイトを訪れた時
- アプリストアから公式サイトにリンクされた時

**優先度**: 最高

## ブランド戦略・UI/UXガイドライン

### WEB-015: ビジュアル・アイデンティティ（幸福の想起）✅ **2026年追加 - ブランド戦略**
**As a** 公式サイトのデザイナー・開発者  
**I want to** 「清潔感（信頼）」と「体温（幸福）」の共存を実現する  
**So that** アプリが「実用的な道具」なら、公式サイトは「ペットと飼い主が紡ぐ物語のショーケース」として機能する

**受け入れ基準**:
- **メインビジュアル（ヒーローセクション）**:
  - 構図：スマートフォンの中でアプリを操作する手元と、その奥でリラックスして眠るペットの「ボケ感」を活かした写真
  - メッセージ：「記録は、愛した証。」のような、情緒的で短いコピーを配置
- **カラーパレット**:
  - アプリのベースカラー（例：柔らかなベージュやパステルブルー）を使いつつ、ウェブでは**「余白（ホワイトスペース）」**をアプリの1.5倍広く取り、情報の呼吸を助ける
- **タイポグラフィ**:
  - 見出しには少し丸みのある、親しみやすいフォントを採用
  - ただし、信頼性を損なわないよう、本文は可読性の高い明朝体やクリーンなゴシック体を使い分ける

**体験価値**:
- **信頼性**: 清潔感のあるデザインにより、アプリの信頼性が向上
- **温かみ**: 幸福を想起させるデザインにより、アプリの温かみが伝わる
- **ブランド性**: アプリと公式サイトで一貫したブランドアイデンティティを確立

**使用シーン**:

  - バッジを付けた誇らしいペットたちのギャラリーや、最新のアップデートニュース

**体験価値**:
- **共感性**: 飼い主の不安に寄り添うことで、共感が生まれる
- **理解の深さ**: 段階的な構成により、アプリの価値を深く理解できる
- **期待感**: 未来のセクションにより、アプリへの期待感が高まる
- **信頼性**: Social Proofにより、アプリの信頼性が向上

- アプリの価値を段階的に理解したい時
