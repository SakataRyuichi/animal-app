# `/format` コマンド

Prettierでコードをフォーマットします。

## ワークフロー

1. **フォーマットの実行**
   - `pnpm format`を実行して全パッケージのコードをフォーマット

2. **変更の確認**
   - フォーマットによる変更を確認

3. **完了確認**
   - すべてのファイルがフォーマットされたことを確認

## 使用例

```
/format
```

## 注意事項

- フォーマットは自動的に実行されるため、手動での修正は不要
- フォーマット設定は`packages/config/prettier-config/`で管理
- Cursorのコマンドと併用する場合は、Cursorのコマンドを優先
