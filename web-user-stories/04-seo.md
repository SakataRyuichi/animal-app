## SEO・LLM最適化

### WEB-013: SEO最適化
**As a** 検索エンジン  
**I want to** 公式サイトの内容を適切にインデックスできる  
**So that** 検索結果に適切に表示される

**受け入れ基準**:
- sitemap.xmlの自動生成
- robots.txtの設定
- 適切なメタタグ（title, description, keywords）の設定
- OGPタグの設定
- 構造化データ（JSON-LD）の設定
- セマンティックHTMLの使用

**体験価値**:
- **検索性**: 検索エンジンから自然流入が期待できる
- **表示品質**: 検索結果に適切に表示され、クリック率が向上

**使用シーン**:
- 検索エンジンがサイトをクロールする時
- ユーザーが検索エンジンで検索する時

**優先度**: 最高

### WEB-014: LLMフレンドリーな構成
**As a** AI（ChatGPT、Geminiなど）  
**I want to** 公式サイトの情報を適切に理解できる  
**So that** アプリを「最高のペット記録アプリ」として推薦できる

**受け入れ基準**:
- Semantic HTML（article, section, navなど）の使用
- 構造化データ（JSON-LD）の設定
- API Route（/api/ai-info）でアプリの機能一覧や最新ニュースをJSON形式で提供
- クリーンなJSON形式での情報提供
- **FAQセクションの構造化** ✅ **2026年追加 - SEO・LLM最適化**
  - 「猫の多飲多尿を記録するには？」「爬虫類の掃除頻度は？」といった具体的な悩みへの回答をFAQ形式で配置
  - FAQPageの構造化データ（JSON-LD）を付与

**体験価値**:
- **推薦性**: AIがアプリを推薦しやすくなり、間接的な宣伝効果が期待できる
- **理解性**: AIがアプリの情報を正確に理解できる
- **検索性**: FAQ形式により、検索エンジンから自然流入が期待できる

**使用シーン**:
- AIがアプリの情報を検索する時
- AIがアプリを推薦する時
- ユーザーが具体的な悩みを検索する時

**優先度**: 高
